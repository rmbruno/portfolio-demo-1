# Clara AI — SaaS Landing Page & Product Demo

Clara AI é um projeto fictício de portfólio que simula uma startup SaaS de inteligência artificial para pequenos negócios, autônomos e equipes comerciais.

## Objetivo
Demonstrar a criação de uma experiência web profissional, responsiva e navegável usando somente HTML5, CSS3 e JavaScript puro.

## Nicho
SaaS de IA para organização comercial, propostas, CRM leve, atendimento e produtividade.

## Tipo
Landing Page SaaS + páginas internas navegáveis + mini-dashboard demonstrativo.

## Funcionalidades
- Landing page completa com hero, benefícios, funcionalidades, prova social, preços, FAQ e CTA.
- Página de recursos detalhada.
- Página de preços com alternador mensal/anual em JavaScript.
- Página de integrações com filtro por categoria.
- Página institucional com missão, valores, equipe e linha do tempo.
- Página de contato com validação visual e mensagem simulada.
- Página de demo com sidebar, métricas, tabelas, proposta gerada por IA simulada, prompt e gráfico visual.
- Menu mobile funcional.
- Contadores animados, FAQ abre/fecha, revelar ao rolar e botão voltar ao topo.

## Como rodar localmente
Abra o arquivo `index.html` diretamente no navegador. Não é necessário instalar dependências nem executar servidor.

## Como publicar no GitHub Pages
1. Envie todos os arquivos para um repositório GitHub.
2. Acesse `Settings > Pages`.
3. Em `Build and deployment`, selecione `Deploy from a branch`.
4. Escolha a branch `main` e a pasta `/root`.
5. Salve e aguarde a URL pública ser gerada.

## Observação
